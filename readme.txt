Customize the WordPress admin User Interface.

== Description ==
Custom Admin UI has the following features:

* Remove the WordPress Logo from the toolbar
* Hiding the Screen Options Tab
* Hide Admin Help Tab
* Customize the Left Footer Text
* Customize the Right Footer Text
* Remove the Admin Color Scheme Picker from the profile page
* Change the page title in admin area

== Installation ==
1. Upload the plugin folder "custom-admin-ui" to the /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins -> Installed plugins -> Custom Admin UI" menu in WordPress.
3. Find "Custom Admin UI" Plugin in -> Settings options.
